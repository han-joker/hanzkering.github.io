Go实现Markdown博客分析设计

  • 1 功能需求
  • 2 技术需求
  • 3 目的
  • 4 效果图

1 功能需求

一个基于 markdown 格式的文档博客系统。提供的功能：

  • markdown 写作
  • 分类
  • 标签
  • 时间归档
  • 配置
  • 友情链接
  • 主题模板
  • 静态化
  • 部署

2 技术需求

  • Go 核心语法
  • Go HTTP服务器搭建
  • HTTP 相关知识
  • 文件管理
  • 正则，字符串处理
  • 时间操作
  • markdown 解析
  • yaml 解析
  • HTML/CSS/Js 前端基本知识

3 目的

通过此项目，对 Go 的基本语法强化练习，会使用第三方包完成具体功能，掌握项目开发流程，github管理代码 等。
